  • Hackations
  • Hackathon 1: Cross-site Scripting Attacks and Defenses
    This Hackathon centers on exploring XSS (Cross-Site Scripting) attacks with the goal of promoting awareness about potential vulnerabilities in web security. There are two main objectives:

    Task 1 (Offensive): The objective is to exploit XSS vulnerabilities in a specified URL (http://waph-hackathon.eastus.cloudapp.azure.com/xss/) through six levels of attack. This requires comprehending the mechanics of XSS attacks, pinpointing vulnerabilities in the provided code, and employing various techniques.

    Task 2 (Defensive): The second part focuses on thwarting XSS attacks by implementing secure coding practices. This involves integrating input validation and output sanitization into the code to prevent the execution of malicious scripts.
